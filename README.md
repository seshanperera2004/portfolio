# Seshan Perera - Portfolio Website

A responsive, animated portfolio website for a Data Science & Analytics professional.

## Features

- Responsive design that works on all devices
- Smooth scrolling animations
- Interactive elements with hover effects
- Navy blue and black color theme
- Four main sections: Home, About, Projects, Contact
- Mobile-friendly navigation menu
- Back-to-top button
- Contact form with validation

## Setup Instructions

1. Create a new folder for your project
2. Add the provided HTML, CSS, and JavaScript files
3. Create an `assets` folder with `images` and `icons` subfolders
4. Add your profile photo and project images to the `assets/images` folder
   - Name them: `profile.jpg`, `project1.jpg`, `project2.jpg`, `project3.jpg`
5. Open `index.html` in your browser to view the website

## Customization

1. Update personal information in `index.html`
2. Replace placeholder images with your own
3. Modify colors in the `:root` section of `style.css`
4. Add more projects by duplicating the project card HTML structure
5. Update social media links in the contact section
